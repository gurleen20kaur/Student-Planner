# AngularProject
## Version 1.0.0

Welcome to the AngularProject repository! This project is built using Angular framework and provides a simple yet effective way to manage daily tasks.
  * Greetings {{username}}! (based on time of day)

Daily To Do's Tab
  + This project is a simple Angular application that allows users to manage their daily to-do lists. Users can add, edit, and delete tasks, as well as mark them as completed.
  + Features:
    + Add new tasks
    + Delete tasks
    + Prioritize tasks (high, medium, low) (color-coded)
    + Reorder tasks via drag-and-drop

Calendar Tab
  + View tasks in a calendar format
  + Navigate between months
  + Click on a date to view tasks for that day
  
Class Schedule Tab
  + View a weekly class schedule
  + Add, edit, and delete classes
  + Color-coded classes based on subject
